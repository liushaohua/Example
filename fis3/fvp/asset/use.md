## 使用方法
### 1、进入fis3以及开启内置服务器.bat
### 2、安装node环境：
```json
	目前用的是Node的v0.12.2版本做的演示。

	安装Fis3：
	npm install -g fis3
	npm install -g fis3-hook-commonjs

	//npm install -g fis-parser-sass
	npm install -g fis-parser-less
	npm install -g fis-parser-template
	npm install -g fis-parser-babel-5.x

	npm install -g fis-postprocessor-autoprefixer

	npm install -g fis3-postpackager-loader

	fis3 install


	3、fis3 server start -p 803 --root D:\work\FVP银翼项目\前端静态资源\fvp\dist

	fis3 release -wL

	dist
	fis3 release dist

停止服务器
fis3 server stop
```
